# Credit Offer Acceptance Prediction

Проект для прогнозирования вероятности согласия корпоративного клиента на кредитное предложение по овердрафту.

Целевая метрика задачи: `ROC-AUC`.

## Суть решения

Итоговая модель обучается на табличных данных о заявке, условиях оффера, характеристиках клиента и его финансовой активности. В решении используются:

- бизнес-признаки по ставке: разница между предложенной ставкой и ключевой ставкой, отношение ставки к ключевой ставке;
- бизнес-признаки по лимитам: соотношение запрошенной суммы с минимальным, максимальным и средним доступным лимитом;
- признаки финансовой активности за 30 и 90 дней;
- логарифмические преобразования денежных величин;
- временные признаки по дате решения;
- признаки региона и последнего продукта клиента;
- OOF target encoding для категориальных признаков;
- ансамбль LightGBM и CatBoost;
- time-based cross-validation.

- `train_apps.csv`
- `test_apps.csv`
- `sample_submission.csv`

## Запуск

Установить зависимости:

```bash
pip install -r requirements.txt
```

Запустить обучение и формирование сабмита:

```bash
python src/train.py
```

После запуска итоговый файл для отправки появится здесь:

```text
submissions/submission.csv
```

## Результат

Текущая итоговая версия решения:

- модель: LightGBM + CatBoost blend;
- локальная валидация: time-based cross-validation;
- средний `ROC-AUC` на локальной валидации: `0.7742117361`;
- файл для загрузки на платформу: `submissions/submission.csv`.
